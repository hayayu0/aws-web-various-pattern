# aws-web-various-pattern

## ここに置いてあるものは何か

AWS環境において、簡易的なWebサーバーを立てる場合に、どのようなパターンがあるのかを考えた時に、以下の8つのサービスが思いつきます。

- EC2
- ECS
- Lightsail
- Elastic Beanstalk
- S3
- Lambda
- ELB
- API Gateway

これがベストプラクティスの8つというわけではなく、他にも方法があると思いますし、
ここに挙げたけれども、ちょっとトリッキーかなという方法も含まれていたりします。

これらの設定に関係するスニペットや設定などを置いています。
